MemFree is a open source <b>Hybrid AI Search Engine</b>.

With MemFree, you could instantly get Accurate Answers from the Internet, Bookmarks, Notes, and Docs.

This documentation is a user guide for MemFree. It will help you to get started with MemFree, and show you how to use MemFree to its full potential.
